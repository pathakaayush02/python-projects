# Python Calculator

This is a simple Python calculator that can perform addition, subtraction, multiplication, and division.

## How to Use
1. Open `calculator.py` in Python.
2. Run the file.
3. Follow the instructions to input numbers and select operations.

## Features
- Add, subtract, multiply, divide
- Works with integers and floats
